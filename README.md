# Image-Forgery
本研究的核心目标是构建一种端到端的深度学习模型，实现对生物医学图像中复制粘贴伪造区域的像素级检测与分割
# 🧪 科学图像伪造检测（Scientific Image Forgery Detection）


本项目聚焦于**科研图像中的复制-粘贴式伪造区域像素级定位**，旨在构建一个高效、轻量且鲁棒的深度学习模型，用于自动检测生物医学等科学论文中常见的图像篡改行为。

---

## 背景与意义

随着图像编辑工具和生成式 AI（如 GAN、扩散模型）的普及，高质量图像伪造变得愈发普遍，严重威胁科研诚信与数字媒体可信度。  
在学术出版领域，**复制-粘贴伪造**（Copy-Move Forgery）是最常见且隐蔽性极强的学术不端行为之一。传统人工审查效率低、主观性强，亟需自动化检测手段。

本项目提出一种基于 **轻量化 U-Net 架构** 的端到端伪造定位方案，专为**类别极度不平衡、伪造区域微小、边界模糊**的科学图像场景设计。

---

## 方法亮点

- **FastUNet 轻量架构**：在标准 U-Net 基础上减少通道数与深度，显著降低计算开销（仅 10.2M 参数，推理时间 ~21ms）。
-  **混合损失函数**：结合 **BCE + Dice Loss**，有效缓解前景像素占比 <0.5% 的极端类别不平衡问题。
- **面向 F1 优化的后处理**：
  - 自适应阈值策略（基于预测均值动态调整）
  - 过滤面积 <100 像素的噪声连通区域
- **端到端训练流程**：从原始图像输入到二值掩码输出，支持快速部署与批量检测。
